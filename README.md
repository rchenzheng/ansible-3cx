Ansible Role: 3CX PBX
=========

Installs 3CX PBX

Requirements
------------

Debian 9 stretch or 10 buster

Example Playbook
----------------

    - hosts: servers
      roles:
         - rchenzheng.3cx_pbx

3CX Setup and Configuration
----------------

Access instance via web browser through port `5015`
